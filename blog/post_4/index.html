4. DOM trong JavaScript

December 24, 2024

—

DOM (Document Object Model) là một khái niệm quan trọng trong lập trình JavaScript, cho phép bạn tương tác và thay đổi nội dung, cấu trúc và kiểu dáng của tài liệu HTML. Nó đóng vai trò là cầu nối giữa mã JavaScript và tài liệu web.

DOM là gì?

DOM là một tiêu chuẩn của W3C (World Wide Web Consortium) định nghĩa cấu trúc tài liệu HTML hoặc XML dưới dạng một cây đối tượng. Mỗi phần tử trong HTML được biểu diễn như một nút (node) trong cây DOM.

Vai trò của DOM trong JavaScript

Thao tác nội dung: Thêm, xóa hoặc thay đổi nội dung của các phần tử HTML. Thay đổi kiểu dáng: Cập nhật CSS của các phần tử để thay đổi giao diện trang web. Xử lý sự kiện: Gắn các sự kiện như click, hover vào các phần tử HTML. Các phương thức phổ biến của DOM Dưới đây là một số phương thức phổ biến mà bạn có thể sử dụng để thao tác với DOM:

document.getElementById(id): Lấy phần tử HTML dựa trên thuộc tính id. document.querySelector(selector): Lấy phần tử đầu tiên phù hợp với bộ chọn CSS. document.querySelectorAll(selector): Lấy tất cả các phần tử phù hợp với bộ chọn CSS. element.innerHTML: Thay đổi hoặc lấy nội dung HTML của phần tử. element.style: Thay đổi kiểu dáng của phần tử. addEventListener(event, function): Gắn sự kiện cho phần tử. Ví dụ sử dụng DOM Đây là một ví dụ đơn giản để thay đổi nội dung của một phần tử HTML bằng DOM:

<!DOCTYPE html>
<html>
<body>
  <h1 id="title">Hello World!</h1>
  <button onclick="changeTitle()">Click me</button>

  <script>
    function changeTitle() {
      document.getElementById("title").innerHTML = "Hello JavaScript!";
    }
  </script>
</body>
</html>

Kết luận

DOM là một phần không thể thiếu trong lập trình JavaScript, giúp bạn xây dựng các ứng dụng web động và tương tác. Bằng cách nắm vững DOM, bạn có thể tùy biến giao diện và chức năng của trang web theo ý muốn.
