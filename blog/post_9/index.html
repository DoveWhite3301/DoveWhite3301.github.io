9. Lập Trình Mạng Máy Tính

December 24, 2024

—

Lập trình mạng máy tính là một lĩnh vực quan trọng trong phát triển phần mềm, giúp các hệ thống có thể giao tiếp và truyền tải dữ liệu giữa các thiết bị với nhau. Bài viết này sẽ giới thiệu các bài toán cơ bản trong lập trình mạng và các ví dụ code mẫu sử dụng các giao thức mạng như TCP và UDP.

Bài Tập 1: Xây Dựng Máy Chủ TCP Đơn Giản

Đây là một bài toán cơ bản để bạn làm quen với việc xây dựng một máy chủ sử dụng giao thức TCP để nhận kết nối từ khách hàng.

import java.io.*;
import java.net.*;

public class SimpleTCPServer {
    public static void main(String[] args) {
        try {
            ServerSocket serverSocket = new ServerSocket(12345);
            System.out.println("Server đang chờ kết nối...");

            Socket clientSocket = serverSocket.accept();
            System.out.println("Kết nối thành công từ: " + clientSocket.getInetAddress());

            InputStreamReader input = new InputStreamReader(clientSocket.getInputStream());
            BufferedReader reader = new BufferedReader(input);
            String message = reader.readLine();
            System.out.println("Nhận được từ client: " + message);

            clientSocket.close();
            serverSocket.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Kết quả khi chạy chương trình sẽ nhận dữ liệu từ một client gửi đến. Dưới đây là một ví dụ về client có thể kết nối đến máy chủ:

import java.io.*;
import java.net.*;

public class SimpleTCPClient {
    public static void main(String[] args) {
        try {
            Socket socket = new Socket("localhost", 12345);
            PrintWriter out = new PrintWriter(socket.getOutputStream(), true);
            out.println("Chào từ client!");

            socket.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Khi chạy máy chủ và client, máy chủ sẽ nhận thông điệp “Chào từ client!” từ client và in ra màn hình.

Bài Tập 2: Xây Dựng Máy Chủ UDP Đơn Giản

UDP là giao thức không kết nối, thích hợp cho các ứng dụng không yêu cầu tính ổn định cao. Dưới đây là cách tạo một máy chủ UDP đơn giản.

import java.net.*;

public class SimpleUDPServer {
    public static void main(String[] args) {
        try {
            DatagramSocket socket = new DatagramSocket(12345);
            byte[] receiveData = new byte[1024];

            while (true) {
                DatagramPacket receivePacket = new DatagramPacket(receiveData, receiveData.length);
                socket.receive(receivePacket);
                String message = new String(receivePacket.getData(), 0, receivePacket.getLength());
                System.out.println("Nhận được từ client: " + message);
            }
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}

Ví dụ client để gửi dữ liệu đến máy chủ UDP:

import java.net.*;

public class SimpleUDPClient {
    public static void main(String[] args) {
        try {
            DatagramSocket socket = new DatagramSocket();
            String message = "Chào từ client UDP!";
            DatagramPacket sendPacket = new DatagramPacket(message.getBytes(), message.length(), InetAddress.getByName("localhost"), 12345);
            socket.send(sendPacket);
            socket.close();
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}

Khi chạy máy chủ và client, máy chủ sẽ nhận thông điệp “Chào từ client UDP!” từ client và in ra màn hình.

Bài Tập 3: Chuyển Dữ Liệu Giữa Các Máy Chủ

Bài tập này yêu cầu bạn tạo một máy chủ trung gian (proxy) để chuyển tiếp các yêu cầu từ client đến một máy chủ đích khác.

import java.io.*;
import java.net.*;

public class ProxyServer {
    public static void main(String[] args) {
        try {
            ServerSocket proxyServerSocket = new ServerSocket(12346);
            System.out.println("Proxy Server đang chờ kết nối...");

            while (true) {
                Socket clientSocket = proxyServerSocket.accept();
                Socket targetServerSocket = new Socket("localhost", 12345);

                InputStreamReader clientInput = new InputStreamReader(clientSocket.getInputStream());
                BufferedReader clientReader = new BufferedReader(clientInput);
                PrintWriter targetWriter = new PrintWriter(targetServerSocket.getOutputStream(), true);

                String clientMessage = clientReader.readLine();
                targetWriter.println(clientMessage);

                BufferedReader targetReader = new BufferedReader(new InputStreamReader(targetServerSocket.getInputStream()));
                String targetMessage = targetReader.readLine();
                System.out.println("Nhận được từ máy chủ đích: " + targetMessage);

                clientSocket.close();
                targetServerSocket.close();
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Trong bài tập này, client sẽ gửi yêu cầu đến proxy server, sau đó proxy server chuyển tiếp yêu cầu đến máy chủ đích và nhận lại dữ liệu từ máy chủ đích để trả về cho client.

Kết Luận

Việc làm quen với lập trình mạng máy tính giúp bạn hiểu cách các hệ thống giao tiếp với nhau qua mạng. Bạn có thể sử dụng các giao thức TCP và UDP để xây dựng các ứng dụng mạng cho nhiều mục đích khác nhau. Bài viết trên giới thiệu một số ví dụ cơ bản về lập trình mạng, và hy vọng sẽ giúp bạn có thêm kiến thức và kỹ năng cần thiết để tiếp cận các dự án lập trình mạng phức tạp hơn trong tương lai.
