3. Các kiểu dữ liệu trong Java

December 24, 2024

—

Trong Java, kiểu dữ liệu là một yếu tố quan trọng giúp định nghĩa cách lưu trữ và thao tác với dữ liệu. Việc hiểu rõ các kiểu dữ liệu trong Java là nền tảng để bạn có thể viết các chương trình chính xác và hiệu quả.

Phân loại kiểu dữ liệu

Java cung cấp hai loại kiểu dữ liệu chính:

Kiểu dữ liệu nguyên thủy (Primitive types): Bao gồm các kiểu cơ bản được Java định nghĩa sẵn. Kiểu dữ liệu đối tượng (Reference types): Bao gồm các đối tượng, mảng và lớp do người dùng định nghĩa. Kiểu dữ liệu nguyên thủy Dưới đây là các kiểu dữ liệu nguyên thủy trong Java:

byte: Lưu trữ số nguyên 8-bit, giá trị từ -128 đến 127. short: Lưu trữ số nguyên 16-bit, giá trị từ -32,768 đến 32,767. int: Lưu trữ số nguyên 32-bit, giá trị từ -231 đến 231-1. long: Lưu trữ số nguyên 64-bit, giá trị từ -263 đến 263-1. float: Lưu trữ số thực 32-bit, phù hợp cho các giá trị số nhỏ. double: Lưu trữ số thực 64-bit, phù hợp cho các giá trị số lớn hoặc có độ chính xác cao. char: Lưu trữ ký tự Unicode 16-bit, dùng để biểu diễn ký tự. boolean: Lưu trữ giá trị đúng (true) hoặc sai (false). Kiểu dữ liệu đối tượng

Các kiểu dữ liệu đối tượng bao gồm:

Mảng: Dùng để lưu trữ danh sách các phần tử cùng kiểu. Chuỗi: Được biểu diễn bởi lớp String, dùng để lưu trữ chuỗi ký tự. Đối tượng: Các đối tượng do người dùng định nghĩa từ các lớp.

Kết luận

Hiểu rõ các kiểu dữ liệu trong Java giúp bạn lựa chọn kiểu phù hợp để tối ưu hóa bộ nhớ và hiệu suất chương trình. Đây là bước đầu tiên trên hành trình học lập trình Java!
